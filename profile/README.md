## How to Install

1. Download the latest version of Tiger Trade for macOS.  
2. Open the DMG installer and move TigerTrade.app to the Applications folder.  
3. Launch the app and allow it through macOS Gatekeeper if required.

By minimizing clutter, Tiger Trade supports a disciplined approach to reading and responding to price.
